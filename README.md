# Curadoria de metadados no OJS

Jogo digital de associação desenvolvido para apoiar uma aula prática sobre Open Journal Systems (OJS), com foco em curadoria de metadados antes da publicação.

A atividade propõe que os estudantes relacionem campos do OJS a ações de conferência editorial. O objetivo é treinar o olhar para a prática: antes de publicar, o registro no OJS precisa estar coerente com o artigo final.

## Objetivo didático

O jogo ajuda os estudantes a reconhecerem pontos de conferência editorial.

A proposta reforça que, no OJS, o artigo também é um registro estruturado, composto por metadados, identificadores e informações editoriais que impactam a recuperação, a circulação, a confiabilidade e a visibilidade científica.

## Como funciona

O jogador deve associar cada **campo do OJS** à **conferência editorial correta**.
